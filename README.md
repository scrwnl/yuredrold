# ゆれどろいど

スマートフォンなどに搭載されている加速度センサの値をバックグラウンドで取得して送りつける

## これはなに

[KusaReMKN/yure](https://github.com/KusaReMKN/yure) の送信機の部分を Android アプリとして書き換えたフォークです。

## つかいかた

1. [Releases](https://github.com/m-tsuru/yuredroid/releases) から最新の apk ファイルをダウンロードする

2. アプリの中央の `Start` ボタンを押下する
